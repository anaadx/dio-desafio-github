- o git é um é um sistema de controle de versões distribuído.
- no prompt de comando no windows:
	- dir: mostra o conteúdo da pasta em que estamos situados.
	- cd: vai para a base do diretório em que estamos situados.
	- \nome da pasta: vai para a pasta digitada.
	- cls: limpa a tela do terminal.
	- TAB: autocompleta a frase no terminal.
	- mkdir: cria nova pasta.
	- del: deleta arquivos na pasta.
	- rmdir: remove pasta.

- principais comandos com o git:
	- git init: iniciar repositório.
	- git add . / git add *: adicionar modificacoes (enviar para staged).
	- git commit -m "comentario": comentario da modificacao - envia novamente para unmodified.
	- git status: mostra status atual do repositorio.
	- git push: envia da maquina para o repositorio.
	- git pull: envia do repositorio para a maquina.
	- git remote add origin "link": adiciona o conteudo ao repositório remoto especificado no link.
